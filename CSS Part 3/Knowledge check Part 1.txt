How do I horizontally center elements using display: flex?
To horizontally center elements within a flex container, apply justify-content: center 

How do I vertically center elements using display: flex?
To vertically center elements within a flex container, apply align-items: center

What is the difference between justify-content and align-items?

-> justify-content aligns items along the main axis (horizontally in a row or vertically in a column), whereas align-items aligns items along the cross axis (vertically in a row or horizontally in a column).
